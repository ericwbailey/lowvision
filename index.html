Low Vision Simulator

Help improve your website's accessibility by simulating common low vision disorders and color-blindness.

Submit the URL of the website you would like to test, then use the dropdown to select your color-blindess or low vision disorder. The simulation may not always be perfect, but should be close enough for evaluation purposes.

Certain websites (Google, Facebook, Flickr, etc.) do not allow embedding, so they will not be able to be evaluated. In order to retain interactivity with the submitted website, this simulator uses SVG filter technology. Unfortunately, this means that only newer browsers may be able to use this site effectively.
